# AnticiFi

**"Know your balance before it happens"**

AnticiFi — это предиктивный AI-ассистент для управления личными финансами. В отличие от классических финансовых приложений, AnticiFi не отслеживает прошлое — он предсказывает будущее состояние вашего счёта.

---

## Миссия

Дать каждому человеку ответ на вопрос: **"Хватит ли мне денег 25-го числа?"** — до того, как этот день наступит.

Мы строим первый в своём классе финансовый инструмент, который смотрит вперёд, а не назад. Вместо того чтобы анализировать, куда ушли деньги в прошлом месяце, AnticiFi использует AI для прогнозирования баланса на любую дату в будущем.

---

## Что умеет AnticiFi

- **Предсказывает баланс** на любую дату на основе анализа паттернов трат и запланированных платежей
- **Импортирует данные** из банковских выписок (CSV) и сканирует чеки через OCR
- **Отслеживает регулярные платежи** — аренда, подписки, коммунальные услуги — и учитывает их в прогнозе
- **Отвечает на вопросы** о будущем балансе в режиме диалога ("Oracle" — предсказательный движок)
- **Работает офлайн** — критические данные хранятся локально на устройстве
- **Уведомляет заранее** о потенциальном недостатке средств до того, как это произошло

## Быстрый старт (для разработчиков)

> Детальные инструкции по деплою и настройке окружения — в [Инфраструктуре и DevOps](./infrastructure.md).

### Предварительные требования

- Docker и Docker Compose
- Node.js 20+
- Flutter 3.x
- Python 3.11+

### Локальный запуск

```bash
# 1. Клонировать репозиторий
git clone https://github.com/anticifi/anticifi.git
cd anticifi

# 2. Скопировать переменные окружения
cp .env.example .env

# 3. Поднять инфраструктуру
docker compose up -d postgres redis nats

# 4. Запустить backend
cd apps/backend
npm install
npm run migration:run
npm run start:dev

# 5. Запустить AI-сервис
cd apps/predictor
pip install -r requirements.txt
uvicorn main:app --reload

# 6. Запустить мобильное приложение
cd apps/mobile
flutter pub get
flutter run
```
